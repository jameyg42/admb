API Key Generator

ADMB can authenticate service requests using a HTTP header (named X-API-KEY) that contains all the information required to log into an AppDynamics controller. NOTE this is a somewhat untraditional "key" since it includes the username/password instead of being an alternative to password!

To help preserve security, the "raw" authentication information is encoded using several different methods (but NOT encrypted!). The form below assists with creating this encoded key.x

X-API-KEY: {{ apiKey }}
